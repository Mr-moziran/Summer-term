## 使用说明

### 前置要求
1. 安装 Node.js (v18+)
2. 安装依赖：`npm install`

### 启动项目
```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:5173

### Mock 开发模式
在后端未完成前，可使用 Apifox Mock Server：
1. 修改 `vite.config.js` 的 proxy 地址
2. 或在 `src/api/*.js` 中临时返回假数据

### 生产构建
```bash
npm run build
```

构建产物在 `dist/` 目录。
