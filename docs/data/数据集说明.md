# 项目演示数据说明

## 当前入库数据

- `demo/src/main/resources/db/seed.sql`：课程演示用中文 SaaS 客服工单数据。覆盖账号、账单、技术、投诉、功能建议等分类，以及全部工单状态、优先级、客服回复、通知和 AI 采纳标记。
- `demo/src/main/resources/db/knowledge-base.json`：300 条课程演示用 FAQ，供用户自助问答检索使用。

以上内容是为本项目业务模型整理的演示数据，不声称来自真实企业，也不含个人信息。

## 公开数据参考

业务工单字段和场景参考 Kaggle 的 [Customer Support Ticket Dataset](https://www.kaggle.com/datasets/suraj520/customer-support-ticket-dataset)。该数据集页面标注为 CC0；若后续直接下载并导入，应保留来源、下载版本、筛选规则和字段映射记录。

## 后续扩容原则

1. 先下载原始公开数据到工作区外或未提交的原始数据目录；不要直接混入 SQL 文件。
2. 删除或匿名化所有可能识别个人的信息。
3. 将英文标题和描述人工校对或翻译为项目的中文 SaaS 场景。
4. 映射分类为 `TECHNICAL`、`BILLING`、`COMPLAINT`、`OTHER`，并映射优先级和状态。
5. 通过审核后再生成可重复执行的项目种子 SQL；FAQ 必须单独维护，不能把未经审核的用户工单直接当作知识库答案。
